What's happening here?

This visualization demonstrates how the scale factor in the positional encoding formula affects sinusoidal wavelengths:

PE(pos, 2i) = sin(pos / (scaleFactor^(2i/d_model)))

Key observations:

  • Smaller scale factors (e.g., 100): Create shorter wavelengths, causing faster oscillations. The pattern repeats more frequently, which limits the ability to represent long sequences without position ambiguity.
  • Larger scale factors (e.g., 100,000): Create longer wavelengths, causing slower oscillations. This allows for representing very long sequences, but may dedicate fewer dimensions to capturing local patterns.
  • The value 10,000: Provides a good balance - long enough to handle typical document lengths but with sufficient coverage of shorter wavelengths for local patterns.
